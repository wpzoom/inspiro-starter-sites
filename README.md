## Description

**Inspiro Starter Sites** is a free plugin designed to simplify the process of building a website with the [Inspiro Lite](https://wordpress.org/themes/inspiro/) theme. It provides access to a library of professionally designed starter templates.

Whether you're creating a blog, an online store, or a portfolio, Inspiro Starter Sites helps you get started with a professional design.

## Features

- **Easy Import** - One-click demo import
- **Pre-designed Layouts** - For blogs, portfolios, e-commerce, and more
- **Gutenberg Compatible** - Works with WordPress block editor
- **Elementor Compatible** - Works with Elementor page builder
- **WooCommerce Ready** - E-commerce templates included

## Installation

### From WordPress.org

1. Go to **Plugins > Add New** in your WordPress admin
2. Search for "Inspiro Starter Sites"
3. Click **Install Now** and then **Activate**

### Manual Installation

1. Download the plugin from [WordPress.org](https://wordpress.org/plugins/inspiro-starter-sites/)
2. Upload to `/wp-content/plugins/inspiro-starter-sites/`
3. Activate through the **Plugins** menu

### From GitHub

```bash
cd wp-content/plugins
git clone https://github.com/wpzoom/inspiro-starter-sites.git
```

## Usage

1. Install and activate the [Inspiro Lite](https://wordpress.org/themes/inspiro/) theme
2. Click on **Starter Sites** button from the theme banner, or go to **Inspiro > Import Demo**
3. Choose the demo you want to use
4. Click **Import Demo** and follow the installer steps

## Requirements

- WordPress 6.0 or higher
- PHP 7.4 or higher
- [Inspiro Lite](https://wordpress.org/themes/inspiro/) theme (recommended)

## FAQ

### Can I use this plugin with themes other than Inspiro?
Inspiro Starter Sites is optimized for use with the Inspiro theme, but it may work with other themes too.

### Are the templates free?
Yes, all templates included in Inspiro Starter Sites are free to use.

### Do I need Elementor or WooCommerce installed?
Templates built for Elementor or WooCommerce require these plugins to be installed and activated.

### Can I customize the imported templates?
Yes, all templates can be customized using either Gutenberg or Elementor, depending on the selected template.
